| 类目                   | 值域               | 备注                     |
|----------------------|------------------|------------------------|
| WeChatFerry-SDK(DLL) | V3.9.X.X         | 每次发布的zip文件包会有版本信息      |
| 配套微信版本               | X.X.X            | 微信安装包的官方版本             |
| 客户端/管理端语言            | JAVA/Python/node | 不同的技术栈会有不同的人维护         |
| 是否测试过其他客户端/管理端       | 是/否              | 由此可以判断是SDK层还是客户端/管理的问题 |
| 具体问题                 | ......           | 描述具体的问题                |
| 复现步骤                 | ......           | 描述出能够复现的步骤             |
